## Waitlist (Supabase)

The waitlist form posts to `POST /api/waitlist`, which inserts into a Supabase table.

### 1) Environment variables

Create `Software/thesis/.env.local`:

- `SUPABASE_URL` (or `NEXT_PUBLIC_SUPABASE_URL`)
- `SUPABASE_SERVICE_ROLE_KEY` (recommended) or `SUPABASE_ANON_KEY`

### 2) Database table

Create a table named `waitlist_signups` in Supabase:

```sql
create table if not exists public.waitlist_signups (
  id bigint generated by default as identity primary key,
  name text,
  email text not null,
	  persona text,
  created_at timestamptz not null default now()
);

create unique index if not exists waitlist_signups_email_unique
  on public.waitlist_signups (lower(email));
```


	If you already created the table, you can add the column with:

	```sql
	alter table public.waitlist_signups
	  add column if not exists persona text;
	```
